# 忘れがちなコマンド

# Insert mode

- `Ctrl-o`: 1回のみノーマルモードのコマンドを打つ

# Normal mode

## 置換

- `R` 固定長のデータ変換

## ヤンク

 - `:%y` バッファ全体のヤンク

## カーソルの位置 を移動する前に戻す

- `<C-o>` 元いた場所に戻る
- `<C-i>` 元いた場所に進む

## 単語

- `gE` 前の単語の末尾へ移動
- `e` 次単語の末尾へ移動

## f{char}

### {char} can be entered as a digraph `digraph-arg`

- `f<C-k>wo` Jump to `を`

※ Not work on `claver-f`

## 行編集

- `cc` 行を削除して，インサートモードに入る
- `gJ` スペースなしの行結合

## スクロール位置の調整
- `zt`  上端へ
- `zz` 真ん中へ
- `zb`  下へ

## マーク機能

`:marks` マーク箇所一覧
`:m[a-zA-Z]` マーキング
`:b [1-99]` 番号を指定しバッファ開く

## surround words

one two -> (one two)

`c2w( Ctrl-R - )`

注 `Ctrl-r`バッファの中身を見る


## Spell check

`<C-x>s` on the underlined word in Insert mode.

## indent

`<C-t>` insert a indent

`<C-d>` delete a indent

## 選択，オペレータ待機モード

- `viw` カーソル上の単語を選択
- `vi"` ダブルクオテーションの内部を選択
- `vit` HTMLタグの内部を選択
- `diw` カーソル上の単語を削除
- `di"` ダブルクオテーションの内部を削除
- `dit` HTMLタグの内部を削除
- `diw` カーソル上の単語を削除してInsert Mode
- `di"` ダブルクオテーションの内部を削除してInsert Mode
- `dit` HTMLタグの内部を削除してInsert Mode
- `d$`  末尾まで削除

### t, f
- `ct_` アンダースコア区切りの変数を変更
- `ct)` 括弧カッコの途中から閉じカッコまでを削除

- `])` 次のマッチするものがない ')' に移動
    - 例:
        - `d])`: `("hello",  args, (1), (2), …any)`途中に複数括弧がある時に便利

注 `di"`でなく`da"`の場合はカッコごと削除

# Visual mode

## 文字数のカウント
- `g Ctrl-g` 文字数のカウント

## CTRL-V
- `Cntl + v + $` 矩形探索，長さが異なる行を全選択
- `Cntl + v + $A` 長さの異なる複数行の行末に同じ文字列を追加する<Plug> conjure

## visual mode

`<localleader>E` evaluate the current selection

`V%<`関数など段落区切りの定義をインデント

## Inserting from register

- `<c-r>"`: insert the last yank/delete
- `<c-r>-`: insert last small (less than a line) delete
- `<c-r>%`: insert the file name
- `<c-r>/`: insert last search terem
- `<c-r>:`: insert last command line

# レジスタ

##  保存
- Normal mode

    1. `"`レジスタを開始する
    2. `a`レジスタを指定
    3. `yy`レジスタに保存

- Visual mode

```txt
V4j"ay
```

## ペースト

- Insert mode

    1. `C-r`
    2. `a`レジスタを指定

- Normal mode

```txt
"ap
```

## レジスタを確認

```txt
:reg a
```

# マクロ

## マクロの保存

`{char}`に保存するマクロを開始

```txt
q{char}
```

## マクロの再生

- `Q`: 最後に記録したマクロを再生する
- `@@`: 最後に再生したマクロを再生する

# <Plug> surrond

## BASICS

- `ds` delete surrounding
- `cs` change surrounding
- `ys` you 'surround'
    - `yss` the line
        - `yssf` add function
    - `ysiw` you surround in a word
    - `ysas` you surround a sentence(文)

## VARIANT

- `yS` own line
- `ySS` own line (effect on entire line)

## VISUAL

- `S`

## LINE VISUAL

- `gS` suppress auto indent

## BLOCK VISUAL

- `gS` `set virtualedit=block`

## INSERT

- `<C-G>s` and `<C-S>` experimental, couldn't get it working myself

## TARGETS

- `), }, ], >`
- `b, B, r, a` aliases
- ` ', ",  `

## MEANING

- `)` no space
- `(` add space between a word and bracket

- `iw` in a word
- `as` a sentence
- `ip` in a paragraph
